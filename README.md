# python-atm-system

This Python program simulates a simple ATM system. It starts by setting an initial account balance of 10000 and a correct PIN of 1234. The user is asked to enter the PIN, and if it matches the correct PIN, the program shows a menu inside a loop so the user can perform multiple operations. The user can choose to check the balance, withdraw money, deposit money, or exit. If the user chooses to check the balance, the program displays the current balance. If they choose to withdraw money, the program asks for an amount and subtracts it from the balance only if there is enough money; otherwise, it shows an insufficient amount message. If the user chooses to deposit money, the program adds the entered amount to the balance only if it is positive. If the user selects exit, the program stops the loop and ends. If the entered PIN is wrong, the program simply prints "Wrong pin".
